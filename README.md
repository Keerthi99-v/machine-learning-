# Beyond Centroids: DBSCAN for Non-Linear Clustering

##  Project Overview
This project explores density-based clustering using DBSCAN to identify non-linear patterns in spatial data.

Unlike centroid-based methods, this work focuses on:
- Discovering clusters of arbitrary shape
- Identifying noise and outliers
- Understanding density-based grouping
- Analysing parameter sensitivity

---

##  Objectives
- Apply DBSCAN to non-linear spatial data
- Select optimal parameters using k-distance graph
- Detect and analyse noise points
- Compare DBSCAN with K-Means clustering
- Evaluate clustering behaviour under different settings

---

##  Dataset
- Synthetic spatial dataset (two-moons structure)
- Generated using Scikit-learn
- Designed to highlight non-linear clustering challenges

---

##  Methods Used

###  DBSCAN
Density-based clustering algorithm that groups points based on local density.

###  k-Distance Graph
Used to determine optimal eps parameter.

###  K-Means (Comparison)
Used to highlight limitations of centroid-based clustering.
